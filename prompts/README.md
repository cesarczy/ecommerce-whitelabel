| Arquivo | Uso |

## Como usar no Cursor

1. Abra o chat do Agent
2. Cole o conteúdo do bloco "Prompt" em `master-prompt.md`
3. Preencha a seção **Contexto do projeto**
4. Envie e acompanhe o progresso spec a spec

## Rules automáticas

As rules em `.cursor/rules/` são aplicadas pelo Cursor conforme os arquivos editados.
Não é necessário referenciá-las manualmente no prompt.
